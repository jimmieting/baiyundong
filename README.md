# 白云洞登山局

> 向上的秩序 · Order Upwards

福州鼓山白云洞攀登计时小程序。从埠兴村登山口出发，到白云洞主洞平台结束，用时自动记录、校验、排名。

## 功能概览

**攀登计时** — 进入起点围栏自动就绪，一键开始，到达终点确认完成。GPS 实时定位 + 地理围栏判定，全程无需手动操作。

**成绩校验** — 云端三重校验（时长合理性、海拔增量、地理轨迹），异常记录自动标记，支持申诉。

**排行榜** — 今日 / 本月 / 历史三维度排名，匿名与实名体系并存。

**分享海报** — 攀登完成后生成带山形剪影的荣誉证书，保存到相册或分享给好友。

**离线容错** — 断网自动缓存，网络恢复后自动同步。支持后台定位、状态检查点恢复。

## 技术栈

- 微信小程序原生框架（WXML + WXSS + JS）
- 微信云开发（云函数 + 云数据库 + 云存储）
- Canvas 2D（计时圆环、海拔折线图、分享海报）
- 无第三方 UI 框架依赖

## 部署步骤

### 1. 导入项目

用微信开发者工具导入本项目，填入 AppID `wx1ae91ef50ebab7bc`。

### 2. 创建云数据库集合

在微信云开发控制台中创建以下集合：

| 集合名 | 说明 | 权限 |
|--------|------|------|
| `t_workout` | 攀登记录 | 仅创建者可写，所有人可读 |
| `t_user` | 用户信息 | 仅创建者可读写 |
| `t_culture` | 文化卡片 | 管理员可写，所有人可读 |
| `t_feedback` | 反馈记录 | 仅创建者可写，管理员可读 |
| `t_cobuilder` | 共建者名录 | 管理员可写，所有人可读 |
| `t_config` | 运营配置 | 管理员可写，所有人可读 |

### 3. 部署云函数

在开发者工具中右键每个云函数目录 → 上传并部署（云端安装依赖）。

### 4. 导入种子数据

部署完成后，在开发者工具中调用 `seedData` 云函数，导入文化卡片和共建者初始数据。

### 5. 运营配置

在 `t_config` 集合中添加配置项：

| key | value | 说明 |
|-----|-------|------|
| `group_qr_url` | 群二维码图片的云存储地址 | 共建页扫码入群 |

### 6. 上线前检查

- 确认 `app.js` 中 `TEST_MODE` 设为 `false`
- 确认服务器域名白名单已配置

## 核心参数

| 参数 | 值 |
|------|-----|
| 起点 | 埠兴村登山口 (26.070797, 119.372559) |
